# AI 图像与真实图像二分类 (AI vs. Real Image Classification)

本项目基于 PyTorch 框架和 ResNet-18 深度残差网络，实现了一个高效的图像二分类系统。核心任务是准确鉴别**真实图像**（标签 `0`）与 **AI 生成图像**（标签 `1`）。

项目包含了完整的数据加载、模型训练、验证评估以及批量推理预测的流程，代码结构已经过规范化重构，方便在任何本地环境或服务器上快速部署和运行。

## 📁 目录结构

建议按照以下结构组织您的项目文件，以确保脚本能够正确找到相对路径：

    AI-Image-Classification/
    ├── data/                   # 数据集目录 (已配置 .gitignore，不上传实际图片)
    │   ├── train/              # 训练集目录
    │   │   ├── 0/              # 存放真实图像 (标签 0)
    │   │   └── 1/              # 存放 AI 生成图像 (标签 1)
    │   └── testset/            # 测试集目录 (存放待预测的无标签图片)
    ├── docs/                   # 项目相关文档
    │   └── 说明文档.docx       # 项目说明与报告
    ├── models/                 # 模型权重保存目录
    │   └── model.pth           # 训练过程中生成的最佳模型权重
    ├── results/                # 预测结果输出目录
    │   └── cla_pre.csv         # 最终生成的测试集预测结果
    ├── src/                    # 源代码目录
    │   ├── train.py            # 模型训练与验证脚本
    │   └── predict.py          # 模型加载与批量预测脚本
    ├── .gitignore              # Git 忽略配置
    ├── README.md               # 项目说明文档
    └── requirements.txt        # Python 依赖库清单

## 🛠️ 环境准备

请确保您的计算机上已安装 Python 3.8 或更高版本。然后在项目根目录下，打开终端（命令行）并运行以下命令安装所需依赖：

    pip install -r requirements.txt

*(依赖包含：torch, torchvision, Pillow, pandas, tqdm)*

## 🗂️ 数据集准备

在运行代码前，请严格按照以下目录结构在项目根目录下准备您的图片数据：

* **训练集**：请在项目根目录下创建 `data/train/` 文件夹。在其中创建两个子文件夹 `0`（存放真实图像）和 `1`（存放 AI 生成图像），并将对应的图片分别放入其中。
* **测试集**：请在项目根目录下创建 `data/testset/` 文件夹。将所有需要预测的未知图片直接放入该文件夹内，无需分子文件夹。

## 🚀 运行模型训练

数据准备完成后，在项目根目录下运行以下命令开始训练模型：

    python src/train.py

**运行结果说明**：
* 终端会显示带有进度条的训练过程，并实时输出每个 Epoch 的 Loss（损失值）和 Accuracy（准确率）。
* 训练完成后，在验证集上表现最好的模型权重文件将被自动保存到 `models/model.pth` 路径下，供后续预测使用。

## 🚀 运行批量预测

确保 `models/` 目录下已经存在训练好的 `model.pth` 权重文件，然后在项目根目录下运行以下命令对测试集进行批量预测：

    python src/predict.py

**运行结果说明**：
* 脚本会自动读取 `data/testset/` 目录下的所有图片并进行推理预测。
* 预测完成后，会自动在 `results/` 目录下生成 `cla_pre.csv` 文件。
* 该 CSV 文件不包含表头，第一列为图像文件名（不含扩展名），第二列为预测的分类结果（`0` 代表真实图像，`1` 代表 AI 生成图像）。

## 🧠 模型架构与策略

* **骨干网络 (Backbone)**: ResNet-18 (基于 ImageNet 预训练权重进行初始化)。
* **分类头 (Classification Head)**: 将全连接层替换为输出维度为 2 的线性层，适配二分类任务。
* **损失函数 (Loss)**: 交叉熵损失函数 (CrossEntropyLoss)。
* **优化器 (Optimizer)**: Adam 优化器 (初始学习率 0.001)。
* **学习率调度 (Scheduler)**: StepLR (每 7 个 epoch 学习率衰减至原来的 10%)。
